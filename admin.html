Публикация каталога на сайт

Сайт по умолчанию показывает не «живую» базу, а последний выгруженный файл catalog.json. После любых изменений товаров нажмите кнопку, скачайте файл и загрузите (замените) его в корне репозитория на GitHub, рядом с index.html.

Синхронизация полного каталога с базой

Одноразовая операция: загружает все товары из catalog.json в Firestore (создаёт недостающие, обновляет фото у существующих). Ничего не удаляет. Нужно один раз нажать после того, как в проект добавлен полный каталог с фотографиями.
